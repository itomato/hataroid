Hataroid v1.606 (20160101)


About

Hataroid is an Android port by squalldc of the open source Atari ST/STE/Falcon emulator Hatari.
Hataroid is released under the GPLv2 http://www.gnu.org/licenses/gpl-2.0.html

This program is and will always be free. If someone charged you for this, please let me know.
